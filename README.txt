=======
Tastemakers
=======

Tastemakers utilities for providing read-only API access.

Version 0.1.5


Introduction
------------
This util permits Tastemakers Africa partners to access and integrate experiences data 
via our Platform APIs.

Errors, comments, questions may be sent to support[at]tastemakersafrica[dot]com.


Requirements
------------
Requires an API keys for read-only calls.

Supported URL Resources
-------------------
- Experiences list
- Single Experience
- Curator profile


Installation
------------
To install, type::

    $ python setup.py install


Troubleshooting
-------------
If you encounter an error while using this sdk, the file
    /tmp/tastemakers_error_log.html
may contain details on what went wrong. In any event, please feel free
to contact support.


Documentation
-------------
Further documentation may be found at https://github.com/TSTMKRSAfrica/Tastemakers-Python-SDK
